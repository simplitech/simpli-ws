# simpli-ws
Kotlin and Java tools for WebService development

## Install
**Gradle**
```
compile group: 'br.com.simpli', name: 'simpli-ws', version: '1.1.1'
```
**Maven**
```xml
<dependency>
  <groupId>br.com.simpli</groupId>
  <artifactId>simpli-ws</artifactId>
  <version>1.1.1</version>
</dependency>
```

## What's inside?

- AwsFileManager
- AWSSendEmailRequest
- AwsSES
- AwsSNS
- ByteArrayDataSource
- PushAndroid
- PushIOS
